# @failure-build/failure-linux-arm64

Platform-specific binary for [`@failure-build/failure`](https://www.npmjs.com/package/@failure-build/failure) on linux-arm64.

Do not install this package directly. Install the main package instead:

```sh
npm install -g @failure-build/failure
```

The main package will automatically pull the correct binary for your platform via `optionalDependencies`.
